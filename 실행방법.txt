VENDIX 실행방법

1. 프로젝트 폴더로 이동
cd "C:\Users\사용자\OneDrive\바탕 화면\web-vending-machine"

2. 가상환경 생성
python -m venv .venv

3. 가상환경 실행
.\.venv\Scripts\activate

4. 필요한 모듈 설치
pip install -r requirements.txt

5. config.py 설정
프로젝트 폴더 안의 config.py 파일에 아래처럼 입력

token = "디스코드 봇 토큰"
GUILD_ID = 123456789012345678

6. 웹서버 실행
PowerShell에서 아래 입력

$env:VENEX_SECRET_KEY="아무 랜덤 긴 문자열"
$env:VENEX_HOST="0.0.0.0"
$env:VENEX_PORT="5000"
python web.py

7. 디스코드 봇 실행
새 PowerShell 창을 하나 더 열고 아래 입력

cd "C:\Users\사용자\OneDrive\바탕 화면\web-vending-machine"
.\.venv\Scripts\activate
python app.py

8. 관리자 페이지 접속
브라우저에서 아래 주소 접속
http://127.0.0.1:5000/login

9. 같은 인터넷의 다른 기기에서 접속
내 컴퓨터 IP를 확인한 뒤 아래처럼 접속
http://내IP:5000/login

10. 새로 시작하는 방법
- 현재 저장소는 개인정보가 없는 초기 상태로 정리됨
- db/database.db 는 비어있는 license 테이블만 들어있음
- 서버별 DB는 없음
- 서버 등록 후 자동으로 새 DB가 생성됨

11. 라이센스 생성
python admin_gen.py

12. 테스트 실행
python -m pytest -q
