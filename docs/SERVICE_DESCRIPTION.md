# VENDIX 서비스 설명

## 1. 서비스 개요

VENDIX는 디스코드 기반 라이센스 등록과 웹 관리자 패널을 결합한 개인 상점(쇼핑몰) 운영 서비스입니다.  
관리자는 서버 단위로 상점을 열고, 사용자 회원가입/로그인, 상품 등록, 구매, 충전 신청 승인까지 한 곳에서 처리할 수 있습니다.

## 2. 핵심 구조

- 봇 서버: `app.py`
  - 라이센스 생성/등록 명령 처리
- 웹 서버: `web.py` (Flask)
  - 관리자 패널, 상점 페이지, 구매/충전/로그 기능 제공
- 데이터베이스: SQLite
  - 공용 DB: `db/database.db` (라이센스 정보)
  - 서버별 DB: `db/<server_id>.db` (상점/유저/상품/로그)

## 3. 주요 기능

- 관리자 기능
  - 기본 설정(상점 공개 여부, 계좌 정보, 로고 등)
  - 유저 관리(잔액, 경고, 차단)
  - 제품 관리(상품 생성/수정/삭제, 재고)
  - 충전 신청 관리(수동 승인, 자동 승인 토글)
  - 라이센스 관리(기간 연장)
  - 감사 로그 조회(`/audit_log`)

- 사용자 기능(상점 페이지)
  - 회원가입/로그인
  - 상품 목록 조회/구매
  - 주문 상세 확인(복사 가능)
  - 충전 신청 및 충전 내역 조회

## 4. 안정성/운영 포인트

- 트랜잭션 락(`BEGIN IMMEDIATE`) 기반으로 재고/잔액/로그를 원자적으로 처리
- 관리자 감사로그로 변경 이력 추적
- 백업/복구 스크립트 제공
  - `scripts/backup_db.py`
  - `scripts/restore_db.py`
- 배포 기본 파일 제공
  - `wsgi.py`, `deploy/nginx.conf.example`, `deploy/.env.example`

## 5. 실행 흐름 (간단)

1. `config.py`에 봇 토큰/길드 ID 입력
2. `python web.py`로 웹 서버 실행
3. `python app.py`로 디스코드 봇 실행
4. 관리자 패널에서 상점 설정/상품 설정
5. 사용자 접속 후 회원가입/구매/충전 이용

## 6. 운영 시 권장 사항

- `VENEX_SECRET_KEY`를 환경변수로 별도 관리
- HTTPS + reverse proxy(Nginx) 사용
- DB 일일 백업 자동화(작업 스케줄러/크론)
- 테스트(`pytest`) 주기 실행
